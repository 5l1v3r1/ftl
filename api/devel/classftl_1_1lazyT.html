Detailed Description

template<typename M>
class ftl::lazyT< M >

The lazy transformer.

Transforms the monad instance M, applied to T, into what is essentially re_parametrise<M,lazy<T>>::type, except the functor series of operations are lifted so that they work on T instead of lazy<T> as they would in the former case.

Concepts
  • FullyConstructible, if the underlying type is.
  • Dereferenceable to the underlying type
  • Functor
  • Applicative Functor
  • Monad
