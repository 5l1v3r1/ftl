Detailed Description

template<typename T>
struct ftl::monoid< std::shared_ptr< T > >

Monoid instance for shared_ptr.

Much like maybe, any shared_ptr that wraps a monoid is also a monoid.

Unwraps the values and applies their monoid op.

Note that if a points to a shared object, but not b, a is returned, not a pointer to a new object that is equal to whatever a points to. Same goes for the revrse situation.

If neither of the pointers point anywhere, another "empty" pointer is returned.

And finally, if both the pointers point to some object, then make_shared is invoked to create a new object that is the result of applying the monoid operation on the two values.
