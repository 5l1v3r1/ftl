Detailed Description

template<>
struct ftl::monad< list >

Monad implementation for list.

Note
This concept instance is only valid for the list alias commonly used in FTL. It is not valid for regular std::lists. If you require the use of an allocator aside from std::allocator, then you must create your own alias of std::list and give it a monad instance.
