Detailed Description

template<typename L, typename M>
class ftl::eitherT< L, M >

The either transformer.

Transforms any given monad, M, so that it also acts like the either monad. In reality, this is done by embedding an ftl::either<L,T> as the concept parameter of M, where T is M's original concept parameter.

In other words, ftl::eitherT<a,std::list<b>> becomes what is basically std::list<ftl::either<a,b>>, except monadic operations on the transform work on b instead of ftl::either<a,b>.

Concepts
  • FullyConstructible, if the transformed type (as mentioned above) is.
  • Assignable, as above.
  • Dereferenceable to the unwrapped transformed type.
  • Functor
  • Applicative Functor
  • Monad
  • Monoidal Alternatives, if L is a Monoid.
  • Foldable, if M is foldable.
Template Parameters
LThe left type in the either values.
MA complete monad type, e.g. std::list<some_type>.
