Detailed Description

template<typename M>
class ftl::maybeT< M >

Maybe Transformer.

This data type transforms any monad M, "stacking" it on top of the maybe monad. The point is to get the functionality of both M and maybe.

Concepts
  • FullyConstructible, if the transformed type is.
  • Assignable, as above.
  • Dereferenceable to the unwrapped transformed type.
  • Functor
  • Applicative Functor
  • Monad
  • monoida
  • Foldable, if M is Foldable.
