Detailed Description

template<typename T>
class ftl::maybe< T >

Abstracts the concept of optional arguments and similar.

In essence, an instance of maybe is either a value, or nothing.

Concepts
Maybe is an instance of the following concepts:
  • FullyConstructible
  • Assignable
  • Dereferenceable
  • EqualityComparable, if, and only if, A is EqComparable
  • orderable, if, and only if, A is Orderable
  • Functor (in A)
  • Applicative Functor (in A)
  • Monad (in A)
  • Monoid, if, and only if, A is a Monoid
  • Foldable
