#include <ftl/concepts/foldable.h>

Function Documentation

template<typename F >
constexpr bool ftl::Foldable ( )
noexcept

Concepts lite-compatible check for foldable instances.

Can also be used with SFINAE, for much the same purposes.

Example:

template<
typename F,
typename = Requires<Foldable<F>()>
>
void foo(const F& f) {
// Perform folds on f
}

struct ftl::ImplementationDefined ftl::fold

Convenience function object for foldable<T>::fold.

Behaves as though it was a function of type

(F<M>) -> M

where M satisfies Monoid.

Example:

list<list<ftl::sum_monoid<int>>> l{
{ftl::sum(1), ftl::sum(2)},
{ftl::sum(3), ftl::sum(4)}
};
auto r = ftl::fmap(ftl::fold, l);
// r == {sum(3), sum(7)}
struct ftl::ImplementationDefined ftl::foldMap

Function object representing foldable::foldMap.

Behaves like a curried function of type

((A) -> M, F<A>) -> M

Makes it generally a lot cleaner to call foldMap, as well as easier to pass to higher order functions.

Examples
auto v = vector<int>{3,3,4};
auto r = ftl::foldMap(ftl::sum<int>, v);
// r == ftl::sum(10)
struct ftl::ImplementationDefined ftl::foldr

Convenience function object for foldable::foldr.

Behaves as if it were a curried function of type

((T,U) -> U, U, F<T>) -> U

Provides easy and concise calling, unlike explicitly looking up the instance implementation as foldable<instance>::foldr.

A right fold is, as the name implies, right associative. Which is to say, the operation foldr(f, z, {1,2,3}) is equivalent to:

f(1, f(2, f(3, z)));
Examples

Summing the elements of a list:

std::list<int> l = {1, 2, 3};
int sum = ftl::foldr(std::plus<int>(), 0, l);
// sum == 6
struct ftl::ImplementationDefined ftl::foldl

Convenience function object representing foldable::foldl.

Behaves as if it were a curried function of type

((U,T) -> U, U, F<T>) -> U

A left fold is of course left-associative, meaning that the operation foldl(f, z, {1,2,3}) is equivalent to:

f(f(f(z, 1), 2), 3);
Examples

Summing the elements of a list:

std::list<int> l = {1, 2, 3};
int sum = ftl::foldl(std::plus<int>(), 0, l);
// sum == 6

Reversing a list:

std::list<int> l = {1, 2, 3};
auto rCons = [](list<int> xs, int x){ xs.push_front(x); return xs; };
auto reverseL = ftl::foldl(rCons, list<int>{}, l);
// reverseL == {3, 2, 1}
