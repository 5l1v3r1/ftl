Detailed Description

template<typename F>
interface ftl::functor< F >

Struct that must be specialised to implement the functor concept.

When specialising, it is important to remember that the described interface here is minimal, and an instance is encouraged to include additional overloads of the functions for optimization reasons and similar. For types that cannot be copied, for instance, it makes sense that all of the functor interface functions accept an rvalue reference.
