Detailed Description

template<typename M>
struct ftl::deriving_join< in_terms_of_bind< M > >

Inheritable implementation of monad::join.

Monad implementations (as in, the template specialisations of monad<>) may inherit this struct to get a default implementation of join, in terms of bind.

Both const reference and r-value reference versions are generated.

Template Parameters
Mthe template specialisation that is to derive join.
Examples
template<typename T>
struct monad<MyMonad<T>>
: deriving_join<in_terms_of_bind<MyMonad<T>>> {
// Implementations of bind, map, apply, and pure
};
Note
Requires that the inheriting monad specialisation implements bind. All of the other monadic operations may be derived.
