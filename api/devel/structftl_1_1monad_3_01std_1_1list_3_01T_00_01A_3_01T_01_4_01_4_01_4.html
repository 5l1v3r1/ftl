Detailed Description

template<typename T, template< typename > class A>
struct ftl::monad< std::list< T, A< T > > >

Monad implementation for std::list.

Note that this instance is only valid for std::lists using an allocator of the form template<typename> class Allocator, where the template parameter to the allocator is the elment type of the list.
