Detailed Description

Run-time sized array container, its concept implementations, and so on.

#include <ftl/vector.h>

Much like ftl::list, this vector type is actually merely a type alias of std::vector. It is used throughout FTL—and recommended for use in applications and libraries using FTL, unless they need a different allocator—because the original vector type's template parameters do not mesh well with e.g. the Functor series of concepts.
