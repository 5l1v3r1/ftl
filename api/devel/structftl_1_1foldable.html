Detailed Description

template<template< typename...> class F>
interface ftl::foldable< F >

Concrete definition of the foldable concept.

The most typical types that are instances of foldable are containers of different sorts, in which case the fold operation is usually a generalisation of a "sum" operation on the elements of the container.
