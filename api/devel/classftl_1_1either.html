Detailed Description

template<typename L, typename R>
class ftl::either< L, R >

Data type modelling a "one of" type.

Put simply, an instance of either<L,R> can store a value of either type L, or type R, but not both at the same time.

Either fulfills the following concepts if, and only if, both of its sub-types also do:

  • MoveConstructible
  • Assignable
  • EqualityComparable

Either fulfills the following concepts regardless of its sub-types:

  • CopyConstructible
  • Dereferenceable
  • EmptyState (when containing an L)
  • Functor (in R)
  • Applicative Functor (in R)
  • Monad (in R)
Note
Either is not DefaultConstructible.
Template Parameters
LThe "left" type, must satisfy CopyConstructible
RThe "right" type , must satisfy CopyConstructible
