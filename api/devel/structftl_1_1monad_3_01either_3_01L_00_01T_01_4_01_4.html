Apply f to right values.

If e is a left value, it's simply passed on without any modification. However, if e is a right value, f is applied and its result is what's passed on.

static

Bind e with the monadic computation f.

If e is a right value, then it's extracted and passed to f, the result of which is the monadic action returned by bind. If e is a left value however, f is never invoked; e.left() is simply passed on.
